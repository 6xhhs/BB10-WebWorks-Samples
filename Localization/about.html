About

This sample application demonstrates different techniques for localizing a BlackBerry 10 WebWorks application:

  • blackberry.system.language provides the value of the current system language.
  • blackberry.event responds to the languagechanged and regionchanged system events.
  • Template.js manages common templates and binds different text values to them.
  • Home screen icons and names can be localized from config.xml.

How to change the language?

  1. Open System settings.
  2. Select 'Language and Input'.
  3. Choose a new language value from the drop-down list box.

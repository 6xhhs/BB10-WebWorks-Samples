Programatic API

A new web view should be created when a button in the action bar is clicked.


window.open(url, '_blank')

This will load a new page in a seperate web view.

window.open(url, '_top')

This will load a new page in this web view.

window.open(url, '_self')

This will load a new page in this web view.

window.open(url, '_parent')

This will load a new page in this web view.

Declarative API

A new web view should be created when a link with target=_blank, _top, _parent, _self is clicked.

target=_blank

The _blank attribute will load a new page in a separate web view.

target=_top

This _top attribute will load a new page in this web view.

target=_self

The _self attribute will load a child page in this web view.

target=_parent

The _parent attribute will load a child page in this web view.
